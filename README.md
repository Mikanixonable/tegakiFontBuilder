<h3>使い方</h3>

<p>p1.pyを実行しtxtファイルから文字表のpng画像を作る </p>
<p>イラストソフト等で文字表を参考に文字を書き、文字表のpng画像に上書き保存</p>

<p>s1.pyを実行し、文字表を文字ごとに分割する</p>

<p>png2ttf.pyで文字の画像からttfフォントをビルドする</p>

<h3>付録</h3>
<p>JIS.txt</p>
<p>常用漢字とかひらがなとかが入っているテキストファイル</p>
